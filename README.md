## 👀 서비스 소개
* 서비스명:  청소 예약서비스 집사

* 서비스설명: 집사는 직고용으로 운영되며 직접운영하는 가사도우미, 이사청소,사무실청소의 예약과
전문업체로 연계 예약하는 침구청소,에어컨청소,펫시팅 으로 나눠 제공하는 서비스 입니다.
